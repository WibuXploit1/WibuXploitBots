## Termux

```bash

> pkg install nodejs

> pkg install libwebp

> pkg install git

> pkg install imagemagick

> pkg install ffmpeg

> termux-setup-storage

> cd /sdcard/(nama file)

> git clone https://github.com/BukanOrangAlay/ershinbotz-v4.git

> cd ershinbotz-v4

> git clone https://github.com/BukanOrangAlay/node_modules
> npm install

> npm start


```
